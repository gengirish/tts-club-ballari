# Demo screenshot output

PNG files in this folder are **generated** by:

```bash
npm run docs:demo-screenshots
```

They are referenced from **`docs/demo-doc-print.html`** and included when you run:

```bash
npm run docs:demo-doc-pdf
```

See **[DEMO_DOC_PDF.md](../DEMO_DOC_PDF.md)** for environment variables and staging vs production behaviour.
